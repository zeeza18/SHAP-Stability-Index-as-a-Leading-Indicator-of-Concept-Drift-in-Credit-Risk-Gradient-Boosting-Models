# Results

| Subdirectory | Contents |
|---|---|
| `tables/` | CSV metrics files (per-window, summary, ablation, SSI, statistical tests) |
| `figures/` | PNG and PDF figures (300 dpi) for the paper |
| `figures/tuning/` | Optuna parameter importance and optimization history plots |
| `checkpoints/` | Saved model files (.joblib) and SHAP arrays (.npy) |

Every number in the paper is traceable to a file in `tables/`.
